# Student Record Management System

## Description
A Java application that manages student records. The system allows users to add, search, update, delete, and display student information. It also supports file handling and report generation.

## Features
- Add Student
- Search Student
- Update Student
- Delete Student
- Display Students
- Save Data to Text File
- Load Data from Text File
- Save Data using Serialization
- Load Serialized Data
- Generate GPA Report

## Technologies Used
- Java
- OOP
- ArrayList
- File Handling
- Serialization
- Exception Handling

## Project Structure

src/
├── Main.java
├── Student.java
├── StudentManager.java
├── FileManager.java
└── ReportGenerator.java

## How to Run

1. Compile the project
2. Run Main.java
3. View the output in the console
